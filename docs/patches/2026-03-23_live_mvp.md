# 2026-03-23 Live MVP Patch

## 개요

이번 패치는 기존 `1:1 영상통화`와 별도로 동작하는 `1:N 유료 라이브` 도메인을 추가하는 작업이다.

목표:
- 관리자 승인형 라이브 권한
- 유료 입장형 라이브
- 관리자 웹 운영 기능
- 앱 내 라이브 탭과 방송방
- 채팅, 선물, mute, kick, chat lock

## 확정 정책

- 개설 권한은 `관리자 수동 활성화`
- 입장료는 `50,000P`
- 유료 시청자 정원은 `10명`
- `admin`은 무료 입장, 정원 제외
- 예정 시간은 `30분 / 1시간`
- 연장은 `30분 / 1시간`, 최대 `2회`
- 환불은 `join ack 실패`만 허용
- 강퇴자는 방 종료 전까지 재입장 금지
- `mute`와 `전체 채팅 잠금` 포함
- 라이브 중 호스트는 `1:1 통화 불가`

## 수정된 주요 오류

- 관리자 헤더가 response에만 실려 admin API에서 읽히지 않던 문제 수정
- 라이브 라우트 일부 문자열/타입 오류 수정
- 관리자 라이브 목록/상세 타입 오류 수정
- 하단 탭 구조에 라이브 탭 추가
- 호스트가 재진입할 때 토큰을 다시 받지 못하던 문제 수정
- 라이브 선물이 기존 `gifts` API와 분리되어 있던 문제 수정

## 남은 실행 전 체크

- 원격 Supabase에 `012_live.sql` 적용
- `live-thumbnails` storage bucket 생성
- `CRON_SECRET` 설정

이번 세션에는 Supabase MCP가 연결되어 있지 않아 원격 DB에 직접 적용하지는 못했고, 마이그레이션 파일과 문서만 준비했다.
