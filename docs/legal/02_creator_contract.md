# wantsome — 크리에이터 용역계약서 전문

> 크리에이터 온보딩 시 전자서명 받는 계약서입니다.
> 서명 완료 시 PDF로 저장되어 private bucket에 보관됩니다.
> 법적 검토 후 사용을 권장합니다.

---

## 콘텐츠 제공 용역계약서

**원썸 컴퍼니(이하 "갑")와 크리에이터(이하 "을")는 다음과 같이 용역계약을 체결합니다.**

---

**제1조 (계약의 목적)**
본 계약은 "갑"이 운영하는 wantsome 플랫폼에서 "을"이 영상통화 콘텐츠를 제공하고, "갑"이 이에 대한 정산을 지급하는 것을 목적으로 합니다.

**제2조 (용역의 내용)**
1. "을"은 wantsome 플랫폼에서 이용자와의 영상통화 서비스를 제공합니다.
2. "을"은 파란불(일반) 모드 및 선택적으로 빨간불(성인) 모드 서비스를 제공할 수 있습니다.
3. "을"은 자신의 일정에 따라 자유롭게 온라인/오프라인 상태를 설정할 수 있습니다.

**제3조 (수익 정산)**
1. 정산 기준: 이용자가 사용한 포인트 중 인앱결제 수수료(30%) 차감 후 실수령액 기준
2. 정산율: 기본 50% (추후 정책 변경 시 별도 고지)
3. 정산일: 매월 15일 (전월 1일~말일 발생 수익 기준)
4. 최소 정산금액: 10,000원 (미달 시 다음 달로 이월)
5. 원천징수: 정산액의 3.3% 공제 후 지급 (사업소득)

**제4조 (정산 등급제)**
서비스 성장 이후 다음 등급제가 적용될 수 있습니다.
- 신규 (가입 후 30일): 55%
- 일반 (월 500분+): 60%
- 인기 (월 1,500분+): 65%
- 탑 (플랫폼 선정): 70%
등급제 도입 시 사전 30일 공지 후 적용합니다.

**제5조 (을의 의무)**
1. "을"은 만 19세 이상 성인임을 보증합니다.
2. "을"은 본인의 의지로 콘텐츠를 제공하며 강요에 의한 것이 아님을 확인합니다.
3. "을"은 이용자와의 영상통화 내용을 동의 없이 녹화·유포하지 않습니다.
4. "을"은 이용자에게 성매매를 유도하거나 직접 만남을 주선하지 않습니다.
5. "을"은 미성년자에게 서비스를 제공하지 않습니다. 미성년자 의심 시 즉시 통화를 종료합니다.
6. "을"은 세금 신고 등 납세 의무를 스스로 이행합니다.

**제6조 (갑의 의무)**
1. "갑"은 정산일에 정산금을 지체 없이 지급합니다.
2. "갑"은 "을"의 개인정보를 안전하게 관리합니다.
3. "갑"은 "을"의 활동을 지원하기 위한 플랫폼 환경을 유지합니다.

**제7조 (금지행위 및 제재)**
다음 행위 발견 시 즉시 계약 해지 및 정산 보류됩니다.
1. 미성년자에게 서비스 제공
2. 성매매 알선 또는 유도
3. 영상 무단 녹화 및 유포
4. 타인 명의 도용

**제8조 (계약 해지)**
1. 양 당사자는 30일 전 서면 통보로 계약을 해지할 수 있습니다.
2. 금지행위 위반 시 즉시 해지됩니다.
3. 계약 해지 시 발생한 정산금은 정산일에 지급됩니다.

**제9조 (면책)**
1. "을"은 독립적인 사업자로서 "갑"과 고용관계가 아님을 확인합니다.
2. "을"의 활동 중 발생한 사고에 대해 "갑"은 책임을 지지 않습니다.

**제10조 (분쟁 해결)**
본 계약과 관련한 분쟁은 서울중앙지방법원을 관할 법원으로 합니다.

## PDF 생성 스펙

```ts
// lib/generateContract.ts
// puppeteer 또는 @react-pdf/renderer 사용

interface ContractData {
  verified_name: string
  birth_date: string
  contract_date: string
  signed_at: string
  ip_address: string
  signature_image: string  // base64
}

// 생성된 PDF → Supabase private bucket
// 경로: contracts/{user_id}/{timestamp}.pdf
// 보관 기간: 계약 유효 기간 동안 영구 보관
```
