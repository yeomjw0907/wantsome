# App Store Connect — 인앱 구입 현지화·심사 문구 (복붙용)

> 서비스: 크리에이터(인플루언서)와 팬이 1:1 영상통화를 이용하는 **wantsome** 앱 내 **소모형 포인트** 충전 상품.  
> **표시 이름·설명**은 아래를 **한국어 현지화**에 그대로 넣으면 됩니다.

---

## 공통 — 심사용 추가 정보 (Review Notes)

아래 블록을 각 인앱 구입의 **심사 정보 → 심사용 추가 정보**에 동일하게 넣어도 됩니다.

```
앱 내 '포인트 충전' 화면에서 본 상품을 선택하면 인앱 결제 플로우가 시작됩니다.
포인트는 크리에이터와의 영상통화·앱 내 유료 기능에 사용되는 가상 재화입니다.
테스트 시 로그인 후 하단 탭에서 충전 메뉴로 이동해 주십시오. (심사용 계정은 App Review 정보에 기재)
```

## 심사 스크린샷

각 인앱 구입마다 **앱에서 해당 상품이 보이는 ‘포인트 충전’ 화면** 캡처를 업로드하면 됩니다. (한 장으로 여러 상품이 보이면 동일 이미지를 재사용 가능한 경우도 있으나, Apple 안내에 따라 상품별로 요구될 수 있음)
