## 과금 플로우 (서버사이드)

```
통화 시작
  └─ 소비자 포인트 ≥ 1분치 단가? NO → "포인트 부족" 팝업
                               YES → Agora 채널 생성 → 통화 시작

통화 중 (Vercel Cron, 1분마다)
  └─ points -= per_min_rate (900 or 1300)
  └─ 잔여 < 5분치 → 클라이언트에 경고 푸시
  └─ 잔여 < 1분치 → 강제 종료

통화 종료
  └─ duration_sec 기록 (초 단위, 분 단위 내림 적용)
  └─ 크리에이터 수익 누적: points_charged × settlement_rate
  └─ call_sessions.status = 'ended'
```

---

## 정산 계산 (매월 15일 Vercel Cron)

```
total_points  = SUM(points_charged) 전월 완료 세션
gross_amount  = total_points × settlement_rate (e.g. 0.5)
tax_amount    = gross_amount × 0.033  ← 원천징수 3.3%
net_amount    = gross_amount - tax_amount
```
