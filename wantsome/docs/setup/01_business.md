# 01. 사업자 등록

> **선행 조건:** 없음
> **소요 시간:** 온라인 신청 30분 + 처리 3~5영업일
> **비용:** 무료
> **왜 필요한가:** PortOne PASS 본인인증, NHN Cloud SMS, 앱스토어 법인/개인사업자 계약, 부가통신사업자 신고 모두 사업자등록번호 필요

---

## Step 1. 사업자 유형 결정

| 유형 | 장점 | 단점 | 권장 상황 |
|------|------|------|-----------|
| **개인사업자** | 빠름, 간단 | 대표자 개인 세금 부담 | 초기 소규모 운영 |
| **법인 (주식회사)** | 투자 유치, 세금 절감 | 설립 비용, 복잡 | 투자 유치 계획 있을 때 |

> wantsome 초기 단계에서는 **개인사업자**로 등록 후 법인 전환 권장

---

## Step 2. 개인사업자 온라인 등록 (홈택스)

### 2-1. 홈택스 접속
1. [hometax.go.kr](https://www.hometax.go.kr) 접속
2. 공동인증서(구 공인인증서) 또는 간편인증(카카오/PASS) 로그인
3. 상단 메뉴 → **사업자등록신청** 클릭

### 2-2. 업종 코드 선택

wantsome에 해당하는 업종 코드:

| 업종 코드 | 업태 | 종목 | 필수 여부 |
|-----------|------|------|-----------|
| **713904** | 정보통신업 | 온라인 플랫폼 운영업 | ✅ 주업종 |
| 743002 | 서비스업 | 전자상거래 소매업 | 포인트/쇼핑 기능 |

> ⚠️ 성인 영상통화 서비스는 **부가통신사업자** 신고도 별도 필요 (→ [09_telecom.md](09_telecom.md))

### 2-3. 입력 정보

```
사업장 주소: 실제 사무실 주소 또는 자택 주소 (공유오피스 가능)
상호명: 원썸 또는 wantsome
대표자명: [본인 이름]
사업 개시일: 오늘 날짜
```

### 2-4. 제출 서류
- 신분증 사본 (온라인 신청 시 스캔 업로드)
- 임대차계약서 (사업장 주소가 자택이 아닐 경우)

### 2-5. 처리 결과 확인
- 신청 후 3~5영업일 내 처리
- 홈택스 → 신청/제출 → 사업자등록신청 → 처리 결과 조회
- 완료 시 **사업자등록증** 발급 (PDF 다운로드)

---

## Step 3. 사업자 등록 후 해야 할 것들

### 3-1. 통장 개설 (사업자 명의)
- 주거래은행에서 **사업자 통장** 개설
- 카카오뱅크/토스뱅크 사업자 계좌도 가능
- PortOne 정산, 크리에이터 정산에 사용

### 3-2. Supabase system_config 업데이트

Supabase Dashboard → SQL Editor에서 실행:

```sql
UPDATE system_config SET value = '원썸' WHERE key = 'company_name';
UPDATE system_config SET value = '[대표자명]' WHERE key = 'ceo_name';
UPDATE system_config SET value = '[000-00-00000]' WHERE key = 'business_number';
UPDATE system_config SET value = '[사업장 주소]' WHERE key = 'business_address';
UPDATE system_config SET value = '[대표 전화번호]' WHERE key = 'customer_service_phone';
```

### 3-3. 이용약관 / 개인정보처리방침 업데이트
- 현재 서버 배포된 약관에 사업자번호, 주소, 대표자명 추가 필요
- `server/app/terms/page.tsx` 하단 사업자 정보 섹션 수정

---

## Step 4. 법인 전환 시 추가 절차 (나중에)

투자 유치 또는 규모 확장 시:

1. 법인 설립 대행 서비스 이용 (스타트업 법무법인, ~50만원)
2. 법인 설립 등기 → 법인 인감 발급
3. 법인 사업자등록증 취득
4. 기존 개인사업자 폐업 신고
